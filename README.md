## my_kernel
A toy kernel I'm writing for fun.

### Notes
- Magic: is a fixed hexadecimal number to identify the head (starting point) of the kernel to be
  loaded.
- Flags: all boot modules loaded
- Checksum: must be sum of magic number and flags
