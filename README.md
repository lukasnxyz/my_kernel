## my_kernel
A 32-bit toy kernel I'm writing for fun.

### Quickstart
Dependency's: `gcc`, `xorriso`, `qemu-system-x86`, `qemu-ui-gtk`

```bash
$ ./build.sh
$ ./run.sh
```
